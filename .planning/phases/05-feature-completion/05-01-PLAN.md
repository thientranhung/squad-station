  <action>
Create `hooks/claude-code-notify.sh` following the exact same guard structure as `hooks/claude-code.sh`, extended to handle the Notification event.

The hook receives this JSON on stdin (Claude Code Notification event):
```json
{
  "hook_event_name": "Notification",
  "notification_type": "permission_prompt",
  "message": "Claude needs permission to use Write",
  "title": "Permission needed",
  "session_id": "...",
  "cwd": "..."
}
```

Script structure (in order):
1. Read stdin fully into NOTIFICATION variable: `NOTIFICATION=$(cat)`
2. Guard: if TMUX_PANE not set, `exit 0`
3. Get AGENT_NAME from tmux: `AGENT_NAME=$(tmux list-panes -t "$TMUX_PANE" -F '#S' 2>/dev/null | head -n1)`; if empty, `exit 0`
4. Guard binary: same `SQUAD_BIN` pattern as `hooks/claude-code.sh`
5. Extract message: `MESSAGE=$(echo "$NOTIFICATION" | python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('message',''))" 2>/dev/null)`; if empty, `exit 0`
6. Find orchestrator session name: `ORCH_NAME=$("$SQUAD_BIN" agents --json 2>/dev/null | python3 -c "import sys,json; agents=json.load(sys.stdin); orch=[a for a in agents if a.get('role')=='orchestrator']; print(orch[0]['name'] if orch else '')" 2>/dev/null)`; if empty, `exit 0`
7. Guard orchestrator session alive: `tmux has-session -t "$ORCH_NAME" 2>/dev/null || exit 0`
8. Forward notification: `tmux send-keys -l -t "$ORCH_NAME" "[NOTIFY] $AGENT_NAME needs permission: $MESSAGE" && tmux send-keys -t "$ORCH_NAME" "" Enter`
9. `exit 0`

File header comment must include registration instructions:
```
# hooks/claude-code-notify.sh -- Forward permission prompt to orchestrator
# Registered under Notification event in .claude/settings.json
# matcher: "permission_prompt"
#
# Register in .claude/settings.json:
#   "hooks": { "Notification": [{ "matcher": "permission_prompt", "hooks": [{ "type": "command", "command": "/path/to/hooks/claude-code-notify.sh" }] }] }
#
# This script ALWAYS exits 0 -- non-zero exits may interfere with provider.
```

Make executable: `chmod +x hooks/claude-code-notify.sh`
