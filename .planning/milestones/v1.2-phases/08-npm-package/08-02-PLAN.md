<task type="auto">
  <name>Task 1: Pack and install locally</name>
  <files>squad-station-0.1.0.tgz</files>
  <action>
Run from the project root:

```bash
npm pack
npm install -g ./squad-station-0.1.0.tgz
```

The postinstall script runs and downloads the correct binary from GitHub Releases. Expected output ends with "squad-station installed successfully".

If npm install fails:
- "HTTP 404": binary naming mismatch or wrong version — check asset names on https://github.com/thientranhung/squad-station/releases
- "permission denied": chmodSync not called — check scripts/postinstall.js line with 0o755
- "Too many redirects": increase redirect limit (currently 5) in postinstall.js

After install, run:
```bash
squad-station --version
which squad-station
```

Record full output of all commands for the SUMMARY.
  </action>

    <automated>which squad-station && squad-station --version</automated>
  </verify>
  <done>npm install -g exits 0 with "squad-station installed successfully". squad-station --version prints version output. which squad-station returns a path.</done>
</task>
